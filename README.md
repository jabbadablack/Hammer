# HAMMER

Hammer is an O3DE gem initially inspired by Valve's Source engine editor. it is meant to view and architech your project from different angles of attack.

## ROADMAP
- [x] View modes. To allow the end user to see their project in a wireframe, quad, shadowmap, heatmap, etc... point of view for maximum control.
- [ ] Vertex, Edge, and Face snapping.
- [ ] Orthogonal toggle.
- [x] Independent game launcher window.
- [x] Dynamic pane layout resizing.
- [x] Ability to maximize select viewpanes.
- [ ] Facing Hotkeys.
    - [ ] Quick snap gizmo.
    - [ ] Focus snapping - Automatic.

## Looking to try it out?

For the time being the development branch of O3DE doesn't have all the small tweaks that are needed to effectively run this gem. here's (my engine fork)[https://github.com/jabbadablack/Seer] if you wish to give it a go.
Please report any bugs, issues, and missing features in the repo.
